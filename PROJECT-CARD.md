- **Estado:** Terminado

- **Template vendible:** SI — plantilla de empresa de servicios con SEO multi-ciudad es muy vendible
- **Mejoras para subir precio:**

  - Chat widget con IA para responder preguntas de clientes 24/7
  - Dashboard de cliente para ver estado de servicios

  - Anadir mas paginas de blog para SEO (objetivo: 20+ articulos)
